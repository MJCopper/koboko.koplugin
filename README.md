# koboko.koplugin
Work-In-Progress KOReader Kobo Plugin

I made this plugin for my own specific use case, it is only tested on my personal Kobo Clara BW. So use it at your own risk.
Thought it would be worth making this code available for others interest.

I use this in conjunction with Grimmory where I store all my books as ePub files and use the Kobo sync feature. I have not tested with other file formats other than ePub.

Do not consider this a completed and tested solution.

Functions of the plugin are;
- Makes kepub folder browsable.
- Setting kepub folder as home.
- Corrects various issues with viewing/opening ePub files from the kepub folder.
- Setting Kobo Sync Server (api_endpoint) from within KOReader - this allows the use of a third party sync servers such as Grimmory.
- Convienant library sync process.
- Collection and series sync.
- Reading progress auto-sync.
